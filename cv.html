My Projects


In this project me with my team created Airline Tickets console application through which users can buy tickets online. We connected our project to the database which will store the data of users who have already registered in the application. In this project we usef different types of classes. One of the main classes is user authentication. We will use it to identify users and determine what data they have access to. There will also be a class for registering users and adding passenger data to my classes. There will also be classes like connectSQL, execSQL, a class for booking tickets, etc. We also used the Singleton pattern to save memory and code usability and many more function.

In this project i create Abstract Factory pattern which can be used for creating cross-platform UI elements without coupling the client code to concrete UI classes, while keeping all created elements consistent with a selected operating system.
